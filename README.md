 This Python script performs exploratory data analysis on the Wisconsin Breast Cancer Dataset to visualize tumor characteristics and diagnosis patterns through six key visualizations: a bar chart showing benign vs malignant tumor counts, a pie chart displaying diagnosis proportions, a heatmap of feature correlations, a ranked bar chart of top predictive features, a histogram of tumor area distribution, and a scatter plot exploring radius-perimeter relationships. The code requires Python 3.x with pandas, numpy, seaborn, and matplotlib installed, and generates six PNG output files saved to the working directory. To use, simply update the CSV file path in the `pd.read_csv()` line and run the script.
